## 4. 代码与配置示例

### 智能告警与自愈脚本

```sh
#!/bin/bash
if grep -q 'ERROR' /var/log/myapp.log; then
  curl -X POST http://alert/api -d '{"msg": "error detected"}'
  systemctl restart myapp
fi
```
