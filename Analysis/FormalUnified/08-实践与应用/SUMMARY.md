```text
         │                       │                       │

└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 数据流设计

1. **需求输入** → 自然语言描述
2. **语义分析** → 概念提取和结构化
3. **模型生成** → 形式化模型创建
4. **可视化** → 图形化表示
5. **形式验证** → 性质检查和验证
6. **代码生成** → 目标语言实现
7. **综合报告** → 完整结果输出
