### 技术规范

#### 数学LaTeX规范

1. **定义格式**
   ```latex
   \begin{definition}[定义名称]
   定义内容
   \end{definition}
   ```

2. **定理格式**
   ```latex
   \begin{theorem}[定理名称]
   定理内容
   \end{theorem}

   \begin{proof}
   证明过程
   \end{proof}
   ```

3. **公理格式**
   ```latex
   \begin{axiom}[公理名称]
   公理内容
   \end{axiom}
   ```

#### 目录结构规范

1. **序号体系**
   - 一级：1.0, 2.0, 3.0, ...
   - 二级：1.1, 1.2, 1.3, ...
   - 三级：1.1.1, 1.1.2, 1.1.3, ...

2. **文件命名**
   - 格式：序号-主题名称.md
   - 示例：01-本体论形式化理论.md

3. **目录命名**
   - 格式：序号-主题名称/
   - 示例：01-哲学基础理论/
