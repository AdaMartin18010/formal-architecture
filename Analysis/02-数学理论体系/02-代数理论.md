## 1. 引言：代数的重要性

### 1.1 代数的定义

**定义 1.1.1** (代数) 代数是研究代数结构及其性质的数学分支。

在形式化框架中，代数可以表示为：

$$\mathcal{A} = (S, \mathcal{O}, \mathcal{R})$$

其中：
- $S$ 是集合 (Set)
- $\mathcal{O}$ 是运算集合 (Operations)
- $\mathcal{R}$ 是关系集合 (Relations)

### 1.2 代数的特征

代数具有以下特征：

1. **抽象性**：研究抽象的结构和性质
2. **普遍性**：适用于多种具体对象
3. **系统性**：具有完整的理论体系
4. **应用性**：在多个领域有重要应用

## 2. 基础概念与定义

### 2.1 代数结构

**定义 2.1.1** (代数结构) 代数结构是一个有序对 $(A, \mathcal{F})$，其中：

- $A$ 是非空集合（载体）
- $\mathcal{F}$ 是运算集合

**定义 2.1.2** (运算) 运算是从集合的笛卡尔积到集合的函数。

$$f: A^n \rightarrow A$$

其中 $n$ 是运算的元数。

**定义 2.1.3** (同态) 两个代数结构之间的同态是保持运算的函数。

$$h: A \rightarrow B \text{ 是同态 } \iff \forall f \in \mathcal{F}: h(f_A(a_1, \ldots, a_n)) = f_B(h(a_1), \ldots, h(a_n))$$

### 2.2 基本性质

**定义 2.2.1** (结合律) 二元运算 $*$ 满足结合律当且仅当：

$$(a * b) * c = a * (b * c)$$

**定义 2.2.2** (交换律) 二元运算 $*$ 满足交换律当且仅当：

$$a * b = b * a$$

**定义 2.2.3** (单位元) 元素 $e$ 是运算 $*$ 的单位元当且仅当：

$$\forall a: e * a = a * e = a$$

**定义 2.2.4** (逆元) 元素 $a$ 的逆元 $a^{-1}$ 满足：

$$a * a^{-1} = a^{-1} * a = e$$

## 3. 群论

### 3.1 群的定义

**定义 3.1.1** (群) 群是一个代数结构 $(G, *)$，满足：

1. **封闭性**：$\forall a, b \in G: a * b \in G$
2. **结合律**：$(a * b) * c = a * (b * c)$
3. **单位元**：$\exists e \in G: \forall a \in G: e * a = a * e = a$
4. **逆元**：$\forall a \in G: \exists a^{-1} \in G: a * a^{-1} = a^{-1} * a = e$

**定理 3.1.1** (单位元唯一性) 群的单位元是唯一的。

**证明** 假设存在两个单位元 $e_1, e_2$：
1. $e_1 = e_1 * e_2 = e_2$
2. 因此单位元唯一

**定理 3.1.2** (逆元唯一性) 群中每个元素的逆元是唯一的。

**证明** 假设元素 $a$ 有两个逆元 $b, c$：
1. $b = b * e = b * (a * c) = (b * a) * c = e * c = c$
2. 因此逆元唯一

### 3.2 子群

**定义 3.2.1** (子群) 群 $(G, *)$ 的子群是 $(H, *)$，其中 $H \subseteq G$ 且 $(H, *)$ 是群。

**定理 3.2.1** (子群判定) 非空子集 $H \subseteq G$ 是子群当且仅当：

$$\forall a, b \in H: a * b^{-1} \in H$$

**证明** 通过子群定义验证：
1. 封闭性：$a * b = a * (b^{-1})^{-1} \in H$
2. 逆元：$a^{-1} = e * a^{-1} \in H$
3. 单位元：$e = a * a^{-1} \in H$

### 3.3 群同态

**定义 3.3.1** (群同态) 群同态是保持群运算的函数。

$$h: G \rightarrow H \text{ 是群同态 } \iff h(a * b) = h(a) * h(b)$$

**定义 3.3.2** (核) 群同态 $h$ 的核是：

$$\ker(h) = \{a \in G \mid h(a) = e_H\}$$

**定理 3.3.1** (同态基本定理) 如果 $h: G \rightarrow H$ 是群同态，则：

$$G / \ker(h) \cong \text{im}(h)$$

### 3.4 特殊群

**定义 3.4.1** (阿贝尔群) 阿贝尔群是满足交换律的群。

$$a * b = b * a$$

**定义 3.4.2** (循环群) 循环群是由单个元素生成的群。

$$G = \langle a \rangle = \{a^n \mid n \in \mathbb{Z}\}$$

**定义 3.4.3** (对称群) 对称群 $S_n$ 是 $n$ 个元素的置换群。

## 4. 环论

### 4.1 环的定义

**定义 4.1.1** (环) 环是一个代数结构 $(R, +, \cdot)$，满足：

1. $(R, +)$ 是阿贝尔群
2. $(R, \cdot)$ 是半群
3. 分配律：$a \cdot (b + c) = a \cdot b + a \cdot c$ 和 $(a + b) \cdot c = a \cdot c + b \cdot c$

**定义 4.1.2** (单位环) 单位环是具有乘法单位元的环。

$$\exists 1 \in R: \forall a \in R: 1 \cdot a = a \cdot 1 = a$$

**定义 4.1.3** (交换环) 交换环是乘法满足交换律的环。

$$a \cdot b = b \cdot a$$

### 4.2 理想

**定义 4.2.1** (理想) 环 $R$ 的理想是子集 $I \subseteq R$，满足：

1. $(I, +)$ 是 $(R, +)$ 的子群
2. $\forall r \in R, a \in I: r \cdot a \in I$ 和 $a \cdot r \in I$

**定义 4.2.2** (主理想) 主理想是由单个元素生成的理想。

$$(a) = \{r \cdot a \mid r \in R\}$$

**定理 4.2.1** (商环) 如果 $I$ 是环 $R$ 的理想，则 $R/I$ 是环。

### 4.3 整环和域

**定义 4.3.1** (整环) 整环是没有零因子的交换环。

$$a \cdot b = 0 \rightarrow a = 0 \lor b = 0$$

**定义 4.3.2** (域) 域是每个非零元素都有乘法逆元的整环。

$$\forall a \neq 0: \exists a^{-1}: a \cdot a^{-1} = 1$$

**定理 4.3.1** (有限整环是域) 有限整环是域。

## 5. 域论

### 5.1 域扩张

**定义 5.1.1** (域扩张) 域扩张是包含关系 $F \subseteq E$。

**定义 5.1.2** (代数扩张) 代数扩张是每个元素都是代数的扩张。

$$\forall \alpha \in E: \exists f(x) \in F[x]: f(\alpha) = 0$$

**定义 5.1.3** (超越扩张) 超越扩张是包含超越元素的扩张。

### 5.2 伽罗瓦理论

**定义 5.2.1** (伽罗瓦群) 域扩张 $E/F$ 的伽罗瓦群是：

$$\text{Gal}(E/F) = \{\sigma: E \rightarrow E \mid \sigma \text{ 是自同构且 } \sigma|_F = \text{id}\}$$

**定理 5.2.1** (伽罗瓦基本定理) 如果 $E/F$ 是有限伽罗瓦扩张，则：

$$\text{Gal}(E/F) \cong \text{Gal}(E/K) / \text{Gal}(E/L)$$

其中 $F \subseteq K \subseteq L \subseteq E$。

## 6. 线性代数

### 6.1 向量空间

**定义 6.1.1** (向量空间) 向量空间是域 $F$ 上的代数结构 $(V, +, \cdot)$，满足：

1. $(V, +)$ 是阿贝尔群
2. 标量乘法：$F \times V \rightarrow V$
3. 分配律和结合律

**定义 6.1.2** (线性无关) 向量集合 $\{v_1, \ldots, v_n\}$ 线性无关当且仅当：

$$\sum_{i=1}^n c_i v_i = 0 \rightarrow c_i = 0 \text{ for all } i$$

**定义 6.1.3** (基) 基是线性无关的生成集。

**定理 6.1.1** (基的存在性) 每个向量空间都有基。

### 6.2 线性变换

**定义 6.2.1** (线性变换) 线性变换是保持线性运算的函数。

$$T: V \rightarrow W \text{ 是线性的 } \iff T(av + bw) = aT(v) + bT(w)$$

**定义 6.2.2** (矩阵表示) 线性变换的矩阵表示是：

$$[T]_\beta^\gamma = [T(v_1)]_\gamma \cdots [T(v_n)]_\gamma$$

**定理 6.2.1** (秩-零化度定理) 如果 $T: V \rightarrow W$ 是线性变换，则：

$$\dim(V) = \text{rank}(T) + \text{nullity}(T)$$

### 6.3 特征值和特征向量

**定义 6.3.1** (特征值) 标量 $\lambda$ 是线性变换 $T$ 的特征值当且仅当：

$$\exists v \neq 0: T(v) = \lambda v$$

**定义 6.3.2** (特征向量) 向量 $v$ 是特征值 $\lambda$ 的特征向量。

**定理 6.3.1** (特征多项式) 特征值是特征多项式的根。

$$\det(T - \lambda I) = 0$$

## 7. 在计算机科学中的应用

### 7.1 密码学

代数理论为密码学提供基础：

```rust
// 代数在密码学中的体现
trait Group {
    type Element;
    
    fn identity(&self) -> Self::Element;
    fn inverse(&self, element: &Self::Element) -> Self::Element;
    fn operation(&self, a: &Self::Element, b: &Self::Element) -> Self::Element;
    fn order(&self) -> Option<usize>;
}

struct CyclicGroup {
    generator: u64,
    modulus: u64,
}

impl Group for CyclicGroup {
    type Element = u64;
    
    fn identity(&self) -> Self::Element {
        1
    }
    
    fn inverse(&self, element: &Self::Element) -> Self::Element {
        // 使用扩展欧几里得算法计算模逆
        self.mod_inverse(*element, self.modulus)
    }
    
    fn operation(&self, a: &Self::Element, b: &Self::Element) -> Self::Element {
        (a * b) % self.modulus
    }
    
    fn order(&self) -> Option<usize> {
        Some(self.modulus as usize - 1)
    }
}

// 椭圆曲线密码学
struct EllipticCurve {
    a: i64,
    b: i64,
    p: u64,
}

impl EllipticCurve {
    fn add(&self, p1: &Point, p2: &Point) -> Point {
        // 实现椭圆曲线点加法
        if p1.is_infinity() {
            return p2.clone();
        }
        if p2.is_infinity() {
            return p1.clone();
        }
        
        if p1.x == p2.x && p1.y == -p2.y {
            return Point::infinity();
        }
        
        let lambda = if p1.x == p2.x {
            // 切线斜率
            let numerator = (3 * p1.x * p1.x + self.a) % self.p as i64;
            let denominator = (2 * p1.y) % self.p as i64;
            (numerator * self.mod_inverse(denominator, self.p as i64)) % self.p as i64
        } else {
            // 割线斜率
            let numerator = (p2.y - p1.y) % self.p as i64;
            let denominator = (p2.x - p1.x) % self.p as i64;
            (numerator * self.mod_inverse(denominator, self.p as i64)) % self.p as i64
        };
        
        let x3 = (lambda * lambda - p1.x - p2.x) % self.p as i64;
        let y3 = (lambda * (p1.x - x3) - p1.y) % self.p as i64;
        
        Point::new(x3, y3)
    }
}
```
