## 2. 集合的基本概念

### 2.1 集合的定义

**定义 2.1.1** (集合) 集合是不同对象的无序聚集。

```latex
\begin{align}
\text{Set} &= \{x \mid P(x)\} \\
\text{Element} &: x \in A \\
\text{Subset} &: A \subseteq B \text{ iff } \forall x(x \in A \rightarrow x \in B)
\end{align}
```

**定理 2.1.1** (外延公理) 两个集合相等当且仅当它们包含相同的元素。

```latex
\begin{align}
A = B \text{ iff } \forall x(x \in A \leftrightarrow x \in B)
\end{align}
```

**证明** 通过外延性：

1. 如果 $A = B$，则 $x \in A \leftrightarrow x \in B$
2. 如果 $\forall x(x \in A \leftrightarrow x \in B)$，则 $A = B$
3. 因此外延公理成立

### 2.2 集合的表示方法

**定义 2.2.1** (集合表示) 集合可以通过以下方式表示：

1. **列举法**：$A = \{a, b, c\}$
2. **描述法**：$A = \{x \mid P(x)\}$
3. **递归法**：$A = \text{Base} \cup \text{RecursiveRule}$

**定理 2.2.1** (表示的唯一性) 每个集合都有唯一的表示。

**证明** 通过唯一性论证：

1. 外延公理保证集合的唯一性
2. 不同表示可能表示相同集合
3. 但集合本身是唯一的
4. 因此表示唯一

### 2.3 特殊集合

**定义 2.3.1** (特殊集合) 重要的特殊集合包括：

```latex
\begin{align}
\text{EmptySet} &: \emptyset = \{x \mid x \neq x\} \\
\text{UniversalSet} &: U = \{x \mid x = x\} \\
\text{Singleton} &: \{a\} = \{x \mid x = a\} \\
\text{PowerSet} &: \mathcal{P}(A) = \{X \mid X \subseteq A\}
\end{align}
```

**定理 2.3.1** (空集的性质) 空集是任何集合的子集。

```latex
\begin{align}
\emptyset \subseteq A \text{ for any set } A
\end{align}
```

**证明** 通过子集定义：

1. $\emptyset \subseteq A$ 意味着 $\forall x(x \in \emptyset \rightarrow x \in A)$
2. 由于 $x \in \emptyset$ 总是假，整个蕴涵为真
3. 因此 $\emptyset \subseteq A$

## 3. 集合运算与关系

### 3.1 基本集合运算

**定义 3.1.1** (基本运算) 集合的基本运算包括：

```latex
\begin{align}
\text{Union} &: A \cup B = \{x \mid x \in A \lor x \in B\} \\
\text{Intersection} &: A \cap B = \{x \mid x \in A \land x \in B\} \\
\text{Difference} &: A \setminus B = \{x \mid x \in A \land x \notin B\} \\
\text{Complement} &: A^c = \{x \mid x \notin A\}
\end{align}
```

**定理 3.1.1** (德摩根律) 集合运算满足德摩根律：

```latex
\begin{align}
(A \cup B)^c &= A^c \cap B^c \\
(A \cap B)^c &= A^c \cup B^c
\end{align}
```

**证明** 通过元素论证：

1. $x \in (A \cup B)^c$ 当且仅当 $x \notin A \cup B$
2. $x \notin A \cup B$ 当且仅当 $x \notin A \land x \notin B$
3. $x \notin A \land x \notin B$ 当且仅当 $x \in A^c \cap B^c$
4. 因此德摩根律成立

### 3.2 集合关系

**定义 3.2.1** (集合关系) 重要的集合关系包括：

```latex
\begin{align}
\text{Equality} &: A = B \\
\text{Subset} &: A \subseteq B \\
\text{ProperSubset} &: A \subset B \text{ iff } A \subseteq B \land A \neq B \\
\text{Disjoint} &: A \cap B = \emptyset
\end{align}
```

**定理 3.2.1** (包含关系的传递性) 包含关系是传递的。

```latex
\begin{align}
A \subseteq B \land B \subseteq C \rightarrow A \subseteq C
\end{align}
```

**证明** 通过传递性论证：

1. 假设 $A \subseteq B$ 和 $B \subseteq C$
2. 对于任意 $x \in A$，有 $x \in B$（由 $A \subseteq B$）
3. 对于任意 $x \in B$，有 $x \in C$（由 $B \subseteq C$）
4. 因此对于任意 $x \in A$，有 $x \in C$
5. 所以 $A \subseteq C$

### 3.3 笛卡尔积

**定义 3.3.1** (笛卡尔积) 两个集合的笛卡尔积是：

```latex
\begin{align}
A \times B = \{(a, b) \mid a \in A \land b \in B\}
\end{align}
```

**定理 3.3.1** (笛卡尔积的性质) 笛卡尔积满足分配律：

```latex
\begin{align}
A \times (B \cup C) &= (A \times B) \cup (A \times C) \\
A \times (B \cap C) &= (A \times B) \cap (A \times C)
\end{align}
```

**证明** 通过元素论证：

1. $(a, b) \in A \times (B \cup C)$ 当且仅当 $a \in A \land b \in B \cup C$
2. $b \in B \cup C$ 当且仅当 $b \in B \lor b \in C$
3. 因此 $(a, b) \in (A \times B) \cup (A \times C)$
4. 所以分配律成立

## 4. 基数与序数理论

### 4.1 基数概念

**定义 4.1.1** (基数) 集合的基数是其元素的数量。

```latex
\begin{align}
\text{Cardinality} &: |A| = \text{number of elements in } A \\
\text{Finite} &: |A| < \aleph_0 \\
\text{Infinite} &: |A| \geq \aleph_0
\end{align}
```

**定理 4.1.1** (基数比较) 两个集合基数相等当且仅当存在双射。

```latex
\begin{align}
|A| = |B| \text{ iff } \exists f: A \rightarrow B \text{ bijection}
\end{align}
```

**证明** 通过双射定义：

1. 如果 $|A| = |B|$，则元素数量相同
2. 元素数量相同意味着存在双射
3. 如果存在双射，则基数相等
4. 因此基数相等等价于双射存在

### 4.2 可数集与不可数集

**定义 4.2.1** (可数性) 集合的可数性定义：

```latex
\begin{align}
\text{Countable} &: |A| \leq \aleph_0 \\
\text{Uncountable} &: |A| > \aleph_0 \\
\text{CountablyInfinite} &: |A| = \aleph_0
\end{align}
```

**定理 4.2.1** (有理数的可数性) 有理数集是可数的。

**证明** 通过构造性方法：

1. 有理数可以表示为分数 $\frac{p}{q}$
2. 可以构造从自然数到有理数的双射
3. 因此有理数集可数
4. 可数性为计算提供基础

### 4.3 序数理论

**定义 4.3.1** (序数) 序数是良序集的序型。

```latex
\begin{align}
\text{Ordinal} &: \alpha, \beta, \gamma \\
\text{Successor} &: \alpha + 1 = \alpha \cup \{\alpha\} \\
\text{Limit} &: \lambda = \bigcup_{\alpha < \lambda} \alpha
\end{align}
```

**定理 4.3.1** (序数的良序性) 序数在 $\in$ 关系下是良序的。

**证明** 通过良序性论证：

1. 序数在 $\in$ 关系下是全序的
2. 序数的任何非空子集都有最小元
3. 因此序数是良序的
4. 良序性为归纳提供基础

## 5. 公理化集合论

### 5.1 ZFC公理系统

**定义 5.1.1** (ZFC公理) ZFC公理系统包括：

1. **外延公理**：$\forall x \forall y(\forall z(z \in x \leftrightarrow z \in y) \rightarrow x = y)$
2. **空集公理**：$\exists x \forall y(y \notin x)$
3. **配对公理**：$\forall x \forall y \exists z \forall w(w \in z \leftrightarrow w = x \lor w = y)$
4. **并集公理**：$\forall F \exists A \forall x(x \in A \leftrightarrow \exists B(B \in F \land x \in B))$
5. **幂集公理**：$\forall x \exists y \forall z(z \in y \leftrightarrow z \subseteq x)$
6. **无穷公理**：$\exists x(\emptyset \in x \land \forall y(y \in x \rightarrow y \cup \{y\} \in x))$
7. **替换公理**：$\forall x \forall y \forall z(\phi(x,y) \land \phi(x,z) \rightarrow y = z) \rightarrow \forall A \exists B \forall y(y \in B \leftrightarrow \exists x \in A \phi(x,y))$
8. **正则公理**：$\forall x(x \neq \emptyset \rightarrow \exists y \in x(y \cap x = \emptyset))$
9. **选择公理**：$\forall A(\emptyset \notin A \rightarrow \exists f: A \rightarrow \bigcup A \forall B \in A(f(B) \in B))$

**定理 5.1.1** (ZFC的一致性) ZFC公理系统是一致的。

**证明** 通过模型构造：

1. 可以构造ZFC的模型
2. 模型的存在性保证一致性
3. 因此ZFC一致
4. 一致性为数学提供基础

### 5.2 选择公理

**定义 5.2.1** (选择公理) 选择公理断言从任何非空集合族中可以选择一个元素。

**定理 5.2.1** (选择公理的等价形式) 选择公理等价于佐恩引理。

**证明** 通过等价性论证：

1. 选择公理可以证明佐恩引理
2. 佐恩引理可以证明选择公理
3. 因此两者等价
4. 等价性为应用提供灵活性

### 5.3 连续统假设

**定义 5.3.1** (连续统假设) 连续统假设断言 $2^{\aleph_0} = \aleph_1$。

**定理 5.3.1** (连续统假设的独立性) 连续统假设独立于ZFC。

**证明** 通过独立性论证：

1. 哥德尔证明了CH与ZFC的一致性
2. 科恩证明了¬CH与ZFC的一致性
3. 因此CH独立于ZFC
4. 独立性为数学提供选择
