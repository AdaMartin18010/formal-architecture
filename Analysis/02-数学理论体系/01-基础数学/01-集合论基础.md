## 1. 概述

### 1.1 集合论定义

**定义 1.1 (集合)**: 集合是不同对象的无序聚集，形式化定义为：

$$S = \{x \mid P(x)\}$$

其中 $P(x)$ 是定义集合的性质。

### 1.2 基本公理

**公理 1.1 (外延公理)**: 两个集合相等当且仅当它们包含相同的元素：

$$\forall A, B, (A = B) \iff (\forall x, x \in A \leftrightarrow x \in B)$$

**公理 1.2 (空集公理)**: 存在一个不包含任何元素的集合：

$$\exists \emptyset, \forall x, x \notin \emptyset$$

**公理 1.3 (配对公理)**: 对于任意两个集合，存在包含它们的集合：

$$\forall a, b, \exists \{a, b\}$$

## 2. 基本概念

### 2.1 集合关系

**定义 2.1 (属于关系)**: 元素属于集合的关系：

$$x \in A$$

**定义 2.2 (包含关系)**: 集合包含关系：

$$A \subseteq B \iff \forall x, x \in A \rightarrow x \in B$$

**定义 2.3 (真包含关系)**: 真包含关系：

$$A \subset B \iff A \subseteq B \land A \neq B$$

### 2.2 集合构造

**定义 2.4 (幂集)**: 集合的所有子集构成的集合：

$$\mathcal{P}(A) = \{B \mid B \subseteq A\}$$

**定义 2.5 (并集)**: 多个集合的并集：

$$\bigcup \mathcal{F} = \{x \mid \exists A \in \mathcal{F}, x \in A\}$$

**定义 2.6 (交集)**: 多个集合的交集：

$$\bigcap \mathcal{F} = \{x \mid \forall A \in \mathcal{F}, x \in A\}$$

## 3. 集合运算

### 3.1 基本运算

**定义 3.1 (并集运算)**: 两个集合的并集：

$$A \cup B = \{x \mid x \in A \lor x \in B\}$$

**定义 3.2 (交集运算)**: 两个集合的交集：

$$A \cap B = \{x \mid x \in A \land x \in B\}$$

**定义 3.3 (差集运算)**: 两个集合的差集：

$$A \setminus B = \{x \mid x \in A \land x \notin B\}$$

**定义 3.4 (对称差集)**: 两个集合的对称差集：

$$A \triangle B = (A \setminus B) \cup (B \setminus A)$$

### 3.2 运算性质

**定理 3.1 (交换律)**: 并集和交集满足交换律：

$$A \cup B = B \cup A$$
$$A \cap B = B \cap A$$

**定理 3.2 (结合律)**: 并集和交集满足结合律：

$$(A \cup B) \cup C = A \cup (B \cup C)$$
$$(A \cap B) \cap C = A \cap (B \cap C)$$

**定理 3.3 (分配律)**: 并集对交集的分配律：

$$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$

**定理 3.4 (德摩根律)**: 德摩根律：

$$(A \cup B)^c = A^c \cap B^c$$
$$(A \cap B)^c = A^c \cup B^c$$

## 4. 关系与函数

### 4.1 关系

**定义 4.1 (二元关系)**: 两个集合之间的二元关系：

$$R \subseteq A \times B$$

**定义 4.2 (等价关系)**: 满足自反、对称、传递的关系：

$$\text{Equivalence}(R) \iff \text{Reflexive}(R) \land \text{Symmetric}(R) \land \text{Transitive}(R)$$

其中：
- $\text{Reflexive}(R) \iff \forall x, (x, x) \in R$
- $\text{Symmetric}(R) \iff \forall x, y, (x, y) \in R \rightarrow (y, x) \in R$
- $\text{Transitive}(R) \iff \forall x, y, z, (x, y) \in R \land (y, z) \in R \rightarrow (x, z) \in R$

### 4.2 函数

**定义 4.3 (函数)**: 满足单值性的关系：

$$f: A \rightarrow B \iff f \subseteq A \times B \land \forall x \in A, \exists! y \in B, (x, y) \in f$$

**定义 4.4 (单射)**: 一对一函数：

$$\text{Injective}(f) \iff \forall x_1, x_2, f(x_1) = f(x_2) \rightarrow x_1 = x_2$$

**定义 4.5 (满射)**: 映上函数：

$$\text{Surjective}(f) \iff \forall y \in B, \exists x \in A, f(x) = y$$

**定义 4.6 (双射)**: 一一对应函数：

$$\text{Bijective}(f) \iff \text{Injective}(f) \land \text{Surjective}(f)$$

## 5. 基数与序数

### 5.1 基数

**定义 5.1 (基数)**: 集合的基数表示其元素个数：

$$|A| = \text{card}(A)$$

**定义 5.2 (等势)**: 两个集合等势当且仅当存在双射：

$$A \approx B \iff \exists f: A \rightarrow B, \text{Bijective}(f)$$

**定义 5.3 (有限集)**: 与某个自然数等势的集合：

$$\text{Finite}(A) \iff \exists n \in \mathbb{N}, A \approx \{1, 2, \ldots, n\}$$

**定义 5.4 (可数集)**: 与自然数集等势的集合：

$$\text{Countable}(A) \iff A \approx \mathbb{N}$$

### 5.2 序数

**定义 5.5 (良序集)**: 每个非空子集都有最小元素的集合：

$$\text{WellOrdered}(A, <) \iff \forall B \subseteq A, B \neq \emptyset \rightarrow \exists \min(B)$$

**定义 5.6 (序数)**: 传递的良序集：

$$\text{Ordinal}(\alpha) \iff \text{Transitive}(\alpha) \land \text{WellOrdered}(\alpha, \in)$$

## 6. 公理化集合论

### 6.1 ZFC公理系统

**公理 6.1 (外延公理)**: 
$$\forall A, B, (\forall x, x \in A \leftrightarrow x \in B) \rightarrow A = B$$

**公理 6.2 (空集公理)**:
$$\exists \emptyset, \forall x, x \notin \emptyset$$

**公理 6.3 (配对公理)**:
$$\forall a, b, \exists \{a, b\}$$

**公理 6.4 (并集公理)**:
$$\forall \mathcal{F}, \exists \bigcup \mathcal{F}$$

**公理 6.5 (幂集公理)**:
$$\forall A, \exists \mathcal{P}(A)$$

**公理 6.6 (无穷公理)**:
$$\exists \omega, \emptyset \in \omega \land \forall x, x \in \omega \rightarrow x \cup \{x\} \in \omega$$

**公理 6.7 (替换公理)**:
$$\forall A, \forall \phi, (\forall x \in A, \exists! y, \phi(x, y)) \rightarrow \exists B, \forall y, y \in B \leftrightarrow \exists x \in A, \phi(x, y)$$

**公理 6.8 (正则公理)**:
$$\forall A \neq \emptyset, \exists x \in A, x \cap A = \emptyset$$

**公理 6.9 (选择公理)**:
$$\forall \mathcal{F}, (\emptyset \notin \mathcal{F} \land \forall A, B \in \mathcal{F}, A \neq B \rightarrow A \cap B = \emptyset) \rightarrow \exists C, \forall A \in \mathcal{F}, |C \cap A| = 1$$

## 7. 形式化证明

### 7.1 基本定理证明

**定理 7.1 (空集唯一性)**: 空集是唯一的：

$$\forall A, (\forall x, x \notin A) \rightarrow A = \emptyset$$

**证明**:
1. 假设 $A$ 是任意空集
2. 根据外延公理，$A = \emptyset$ 当且仅当 $\forall x, x \in A \leftrightarrow x \in \emptyset$
3. 由于 $A$ 和 $\emptyset$ 都不包含任何元素，条件成立
4. 因此 $A = \emptyset$
5. 证毕

**定理 7.2 (幂集性质)**: 幂集的基本性质：

$$\forall A, \emptyset \in \mathcal{P}(A) \land A \in \mathcal{P}(A)$$

**证明**:
1. $\emptyset \subseteq A$ 恒成立
2. 因此 $\emptyset \in \mathcal{P}(A)$
3. $A \subseteq A$ 恒成立
4. 因此 $A \in \mathcal{P}(A)$
5. 证毕

### 7.2 基数运算

**定理 7.3 (基数不等式)**: 对于有限集：

$$|A \cup B| = |A| + |B| - |A \cap B|$$

**证明**:
1. 使用容斥原理
2. $|A \cup B| = |A| + |B| - |A \cap B|$
3. 证毕

## 8. 应用与实现

### 8.1 集合数据结构

**定义 8.1 (集合实现)**: 集合的计算机实现：

```rust
pub struct Set<T: Eq + Hash> {
    elements: HashMap<T, ()>,
}

impl<T: Eq + Hash> Set<T> {
    pub fn new() -> Self {
        Set {
            elements: HashMap::new(),
        }
    }
    
    pub fn insert(&mut self, element: T) {
        self.elements.insert(element, ());
    }
    
    pub fn contains(&self, element: &T) -> bool {
        self.elements.contains_key(element)
    }
    
    pub fn union(&self, other: &Set<T>) -> Set<T> {
        let mut result = self.clone();
        for element in other.elements.keys() {
            result.insert(element.clone());
        }
        result
    }
    
    pub fn intersection(&self, other: &Set<T>) -> Set<T> {
        let mut result = Set::new();
        for element in self.elements.keys() {
            if other.contains(element) {
                result.insert(element.clone());
            }
        }
        result
    }
}
```

### 8.2 关系实现

**定义 8.2 (关系实现)**: 二元关系的实现：

```rust
pub struct Relation<T: Eq + Hash> {
    pairs: HashSet<(T, T)>,
}

impl<T: Eq + Hash> Relation<T> {
    pub fn new() -> Self {
        Relation {
            pairs: HashSet::new(),
        }
    }
    
    pub fn add_pair(&mut self, a: T, b: T) {
        self.pairs.insert((a, b));
    }
    
    pub fn is_reflexive(&self) -> bool {
        // 检查所有元素是否与自己有关系
        for (a, b) in &self.pairs {
            if a == b {
                return true;
            }
        }
        false
    }
    
    pub fn is_symmetric(&self) -> bool {
        for (a, b) in &self.pairs {
            if !self.pairs.contains(&(b.clone(), a.clone())) {
                return false;
            }
        }
        true
    }
    
    pub fn is_transitive(&self) -> bool {
        for (a, b) in &self.pairs {
            for (c, d) in &self.pairs {
                if b == c && !self.pairs.contains(&(a.clone(), d.clone())) {
                    return false;
                }
            }
        }
        true
    }
}
```

### 8.3 函数实现

**定义 8.3 (函数实现)**: 数学函数的实现：

```rust
pub struct Function<D: Eq + Hash, R> {
    mapping: HashMap<D, R>,
    domain: Set<D>,
    codomain: Set<R>,
}

impl<D: Eq + Hash, R: Clone> Function<D, R> {
    pub fn new(domain: Set<D>, codomain: Set<R>) -> Self {
        Function {
            mapping: HashMap::new(),
            domain,
            codomain,
        }
    }
    
    pub fn apply(&self, x: &D) -> Option<&R> {
        self.mapping.get(x)
    }
    
    pub fn is_injective(&self) -> bool {
        let mut seen = HashSet::new();
        for value in self.mapping.values() {
            if seen.contains(value) {
                return false;
            }
            seen.insert(value);
        }
        true
    }
    
    pub fn is_surjective(&self) -> bool {
        for element in &self.codomain.elements {
            let mut found = false;
            for value in self.mapping.values() {
                if value == element.0 {
                    found = true;
                    break;
                }
            }
            if !found {
                return false;
            }
        }
        true
    }
    
    pub fn is_bijective(&self) -> bool {
        self.is_injective() && self.is_surjective()
    }
}
```
