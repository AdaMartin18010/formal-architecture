## 1. 概述

### 1.1 基础数学定义

基础数学是现代数学的基石，为所有数学分支提供基本概念、符号和推理方法。在形式化架构理论中，基础数学为系统建模、逻辑推理和结构分析提供数学基础。

```latex
\text{基础数学} = \{\text{集合论}, \text{数系}, \text{逻辑}, \text{关系}, \text{函数}, \text{代数结构}\}
```

### 1.2 基础数学在架构理论中的作用

1. **建模基础**：为系统建模提供数学语言
2. **推理基础**：为逻辑推理提供形式化方法

## 2. 集合论基础

### 2.1 集合基本概念

#### 2.1.1 集合定义
```latex
\text{集合} = \{x \mid P(x)\}
```

其中 $P(x)$ 是描述集合元素性质的谓词。

#### 2.1.2 集合运算
```latex
\begin{align}
A \cup B &= \{x \mid x \in A \lor x \in B\} \\
A \cap B &= \{x \mid x \in A \land x \in B\} \\
A \setminus B &= \{x \mid x \in A \land x \notin B\} \\
A^c &= \{x \mid x \notin A\}
\end{align}
```

### 2.2 集合关系

#### 2.2.1 包含关系
```latex
\begin{align}
A \subseteq B &\leftrightarrow \forall x (x \in A \rightarrow x \in B) \\
A = B &\leftrightarrow A \subseteq B \land B \subseteq A
\end{align}
```

#### 2.2.2 幂集
```latex
\mathcal{P}(A) = \{B \mid B \subseteq A\}
```

### 2.3 集合公理系统

#### 2.3.1 ZFC公理
```latex
\begin{align}
\text{外延公理: } & \forall x \forall y (\forall z (z \in x \leftrightarrow z \in y) \rightarrow x = y) \\
\text{空集公理: } & \exists x \forall y (y \notin x) \\
\text{配对公理: } & \forall x \forall y \exists z \forall w (w \in z \leftrightarrow w = x \lor w = y) \\
\text{并集公理: } & \forall F \exists A \forall x (x \in A \leftrightarrow \exists B (B \in F \land x \in B))
\end{align}
```

## 3. 数系理论

### 3.1 自然数

#### 3.1.1 皮亚诺公理
```latex
\begin{align}
\text{P1: } & 0 \in \mathbb{N} \\
\text{P2: } & \forall n (n \in \mathbb{N} \rightarrow S(n) \in \mathbb{N}) \\
\text{P3: } & \forall n (S(n) \neq 0) \\
\text{P4: } & \forall m \forall n (S(m) = S(n) \rightarrow m = n) \\
\text{P5: } & \forall A ((0 \in A \land \forall n (n \in A \rightarrow S(n) \in A)) \rightarrow \mathbb{N} \subseteq A)
\end{align}
```

#### 3.1.2 递归定义
```latex
\begin{align}
\text{加法: } & m + 0 = m, \quad m + S(n) = S(m + n) \\
\text{乘法: } & m \cdot 0 = 0, \quad m \cdot S(n) = m \cdot n + m
\end{align}
```

### 3.2 整数

#### 3.2.1 整数构造
```latex
\mathbb{Z} = \mathbb{N} \times \mathbb{N} / \sim
```

其中等价关系 $\sim$ 定义为：
```latex
(a,b) \sim (c,d) \leftrightarrow a + d = b + c
```

#### 3.2.2 整数运算
```latex
\begin{align}
[(a,b)] + [(c,d)] &= [(a+c, b+d)] \\
[(a,b)] \cdot [(c,d)] &= [(ac+bd, ad+bc)]
\end{align}
```

### 3.3 有理数

#### 3.3.1 有理数构造
```latex
\mathbb{Q} = \mathbb{Z} \times (\mathbb{Z} \setminus \{0\}) / \sim
```

其中等价关系 $\sim$ 定义为：
```latex
(a,b) \sim (c,d) \leftrightarrow a \cdot d = b \cdot c
```

#### 3.3.2 有理数性质
```latex
\begin{align}
\text{稠密性: } & \forall p \forall q (p < q \rightarrow \exists r (p < r < q)) \\
\text{可数性: } & |\mathbb{Q}| = \aleph_0
\end{align}
```

### 3.4 实数

#### 3.4.1 戴德金分割
```latex
\mathbb{R} = \{(A,B) \mid A,B \subseteq \mathbb{Q} \land \text{分割}(A,B)\}
```

其中分割条件：
```latex
\text{分割}(A,B) \leftrightarrow \begin{cases}
A \neq \emptyset \land B \neq \emptyset \\
A \cup B = \mathbb{Q} \\
A \cap B = \emptyset \\
\forall a \in A \forall b \in B (a < b)
\end{cases}
```

#### 3.4.2 实数性质
```latex
\begin{align}
\text{完备性: } & \text{每个有上界的非空集合有最小上界} \\
\text{不可数性: } & |\mathbb{R}| = 2^{\aleph_0}
\end{align}
```

### 3.5 复数

#### 3.5.1 复数定义
```latex
\mathbb{C} = \mathbb{R} \times \mathbb{R}
```

#### 3.5.2 复数运算
```latex
\begin{align}
(a,b) + (c,d) &= (a+c, b+d) \\
(a,b) \cdot (c,d) &= (ac-bd, ad+bc)
\end{align}
```

\phi \rightarrow \psi & \text{ (蕴含)} \\

```

#### 4.1.2 命题逻辑公理

\text{A2: } & (\phi \rightarrow (\psi \rightarrow \chi)) \rightarrow ((\phi \rightarrow \psi) \rightarrow (\phi \rightarrow \chi)) \\
\text{A3: } & (\neg \phi \rightarrow \neg \psi) \rightarrow (\psi \rightarrow \phi)

```

\end{align}

## 5. 关系与函数

### 5.1 关系

#### 5.1.1 关系定义
```latex
R \subseteq A \times B
```

#### 5.1.2 关系性质
```latex
\begin{align}
\text{自反性: } & \forall x (xRx) \\
\text{对称性: } & \forall x \forall y (xRy \rightarrow yRx) \\
\text{传递性: } & \forall x \forall y \forall z (xRy \land yRz \rightarrow xRz) \\
\text{反对称性: } & \forall x \forall y (xRy \land yRx \rightarrow x = y)
\end{align}
```

### 5.2 等价关系

#### 5.2.1 等价关系定义
```latex
\text{等价关系} = \{\text{自反性}, \text{对称性}, \text{传递性}\}
```

#### 5.2.2 等价类
```latex
[a] = \{x \mid x \sim a\}
```

#### 5.2.3 商集
```latex
A/\sim = \{[a] \mid a \in A\}
```

### 5.3 函数

#### 5.3.1 函数定义
```latex
f: A \rightarrow B = \{(a,b) \mid a \in A \land b \in B \land \text{单值}(f,a,b)\}
```

其中单值性：
```latex
\text{单值}(f,a,b) \leftrightarrow \forall c ((a,c) \in f \rightarrow c = b)
```

#### 5.3.2 函数性质
```latex
\begin{align}
\text{单射: } & \forall x \forall y (f(x) = f(y) \rightarrow x = y) \\
\text{满射: } & \forall y \exists x (f(x) = y) \\
\text{双射: } & \text{单射} \land \text{满射}
\end{align}
```

## 6. 代数结构基础

### 6.1 群

#### 6.1.1 群定义
```latex
\text{群} = \langle G, \cdot, e, ^{-1} \rangle
```

满足公理：
```latex
\begin{align}
\text{结合律: } & \forall x \forall y \forall z ((x \cdot y) \cdot z = x \cdot (y \cdot z)) \\
\text{单位元: } & \forall x (e \cdot x = x \cdot e = x) \\
\text{逆元: } & \forall x (x \cdot x^{-1} = x^{-1} \cdot x = e)
\end{align}
```

#### 6.1.2 子群
```latex
H \leq G \leftrightarrow H \subseteq G \land \text{群}(H)
```

### 6.2 环

#### 6.2.1 环定义
```latex
\text{环} = \langle R, +, \cdot, 0, 1 \rangle
```

满足公理：
```latex
\begin{align}
\text{加法群: } & \langle R, +, 0 \rangle \text{是阿贝尔群} \\
\text{乘法半群: } & \langle R, \cdot, 1 \rangle \text{是幺半群} \\
\text{分配律: } & \forall x \forall y \forall z (x \cdot (y + z) = x \cdot y + x \cdot z)
\end{align}
```

### 6.3 域

## 7. 序理论

### 7.1 偏序

#### 7.1.1 偏序定义
```latex
\text{偏序} = \langle P, \leq \rangle
```

满足公理：
```latex
\begin{align}
\text{自反性: } & \forall x (x \leq x) \\
\text{反对称性: } & \forall x \forall y (x \leq y \land y \leq x \rightarrow x = y) \\
\text{传递性: } & \forall x \forall y \forall z (x \leq y \land y \leq z \rightarrow x \leq z)
\end{align}
```

#### 7.1.2 全序
```latex
\text{全序} = \text{偏序} \land \forall x \forall y (x \leq y \lor y \leq x)
```

### 7.2 格

#### 7.2.1 格定义
```latex
\text{格} = \langle L, \wedge, \vee \rangle
```

满足公理：
```latex
\begin{align}
\text{交换律: } & x \wedge y = y \wedge x, \quad x \vee y = y \vee x \\
\text{结合律: } & (x \wedge y) \wedge z = x \wedge (y \wedge z) \\
\text{吸收律: } & x \wedge (x \vee y) = x, \quad x \vee (x \wedge y) = x
\end{align}
```

## 8. 基数与序数

### 8.1 基数

#### 8.1.1 基数定义
```latex
\text{基数} = \{\alpha \mid \alpha \text{是序数} \land \forall \beta < \alpha (|\beta| < |\alpha|)\}
```

#### 8.1.2 基数运算
```latex
\begin{align}
|\alpha| + |\beta| &= |\alpha \sqcup \beta| \\
|\alpha| \cdot |\beta| &= |\alpha \times \beta| \\
|\alpha|^{|\beta|} &= |\alpha^{\beta}|
\end{align}
```
