## 2. 简单类型论

### 2.1 简单类型 λ 演算

**定义 2.1** (简单类型)

$$\tau ::= \mathbf{1} \mid \tau \rightarrow \tau \mid \tau \times \tau$$

- $\mathbf{1}$：单位类型
- $\tau \rightarrow \sigma$：函数类型
- $\tau \times \sigma$：积类型

**定义 2.2** (项)

$$M ::= x \mid \lambda x:\tau.M \mid M\,N \mid (M,N) \mid \pi_1(M) \mid \pi_2(M)$$

### 2.2 类型规则

$$\frac{}{\Gamma, x:\tau \vdash x : \tau}$$

$$\frac{\Gamma, x:\tau \vdash M : \sigma}{\Gamma \vdash \lambda x:\tau.M : \tau \rightarrow \sigma}$$

$$\frac{\Gamma \vdash M : \tau \rightarrow \sigma \quad \Gamma \vdash N : \tau}{\Gamma \vdash M\,N : \sigma}$$

## 3. 依赖类型论

### 3.1 Martin-Löf 类型论

**依赖积类型**（Π 类型）：$\Pi_{x:A} B(x)$，项的类型依赖于值。

**依赖和类型**（Σ 类型）：$\Sigma_{x:A} B(x)$，存在型。

### 3.2 与形式化证明

- **命题即类型**：命题 $P$ 对应类型，证明对应类型 $P$ 的项
- **结构同构**：可形式化为依赖类型 $f : \Pi_{e:M_{sem}} M_{tech}$

## 4. Curry-Howard 对应

| 逻辑 | 类型论 |
|------|--------|
| 命题 | 类型 |
| 证明 | 项 |

| 析取 $A \lor B$ | 和类型 $A + B$ |
