## 三、文档交叉引用规范

### 3.1 Modern → Analysis

- **路径格式**：`../../Analysis/<目录>/<文件>.md`
- **示例**：从 `Modern/01-IT语义世界基础理论/02-MSMFIT.md` 引用 `Analysis/01-哲学基础理论/`

### 3.2 Analysis → Modern

- **路径格式**：`../../Modern/<目录>/<文件>.md`
- **示例**：从 `Analysis/07-理论统一与整合/` 引用 `Modern/02-语义驱动架构理论/`

### 3.3 Modern 内部

- **路径格式**：`../<同级目录>/` 或 `./<同目录文件>`
- **示例**：`../09-理论增强与完善/01-形式化证明增强/03-MSMFIT最小性的不可约简性证明.md`
