# 主题系统的代数：暗色模式与动态令牌

> **来源映射**: View/05.md §6.1
> **权威参考**: CSS Custom Properties, OKLCH 色彩空间

---

## 一、核心概念

```text
主题切换的形式化:
  设主题空间 Θ = {light, dark, high-contrast, ...}
  设令牌函数 token: Θ × T → CSSValue

  暗色模式: token(dark, "background") = #000
  亮色模式: token(light, "background") = #fff

  动态令牌: 通过 CSS 变量在运行时切换
    --background: var(--theme-background);
    通过根元素 class 切换主题上下文
```
