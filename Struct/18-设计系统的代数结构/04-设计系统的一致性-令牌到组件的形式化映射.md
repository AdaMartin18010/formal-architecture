# 设计系统的一致性：令牌到组件的形式化映射

> **来源映射**: View/05.md §6
> **权威参考**: Design Systems Handbook (InVision)

---

## 一、核心概念

```text
映射操作 ⊗: T × C → StyledComponent
  输入: 设计令牌 t ∈ T, 组件 c ∈ C
  输出: 样式化组件 s = t ⊗ c

一致性约束:
  ∀t₁, t₂ ∈ T, ∀c ∈ C:
    t₁ ≡ t₂ ⇒ t₁ ⊗ c ≡ t₂ ⊗ c

  即: 等价令牌应产生等价样式
```
