## 一、定理 T7：设计令牌构成阿贝尔群

```text
定义 (Join-Semilattice 扩展): 𝓓 = ⟨T, ⊕, ⊗, 𝟎, 𝟏⟩

封闭性: ∀a,b ∈ T, a ⊕ b ∈ T
  例: color(#000) ⊕ opacity(0.5) = color(rgba(0,0,0,0.5))

结合律: (a ⊕ b) ⊕ c = a ⊕ (b ⊕ c)
  例: (spacing(4) ⊕ 8) ⊕ 16 = 4 ⊕ (8 ⊕ 16) = 28

交换律: a ⊕ b = b ⊕ a

单位元: 𝟎 = transparent, spacing(0)

逆元: spacing(8)⁻¹ = spacing(-8)

∴ ⟨T, ⊕⟩ 是阿贝尔群    ∎

应用: CSS 变量 (Custom Properties) 是群的线性表示
  --color-primary: #000;
  这些变量是群 T 的生成元 (Generators)
```
