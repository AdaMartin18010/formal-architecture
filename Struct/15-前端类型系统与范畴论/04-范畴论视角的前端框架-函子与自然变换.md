## 一、核心概念定义

### 1.1 前端渲染范畴

```text
定义 (前端渲染范畴 𝓒):
  𝓒 = ⟨Obj(𝓒), Hom(𝓒), ∘, id⟩

  Obj(𝓒): {State, VDOM, DOM, Effect, Component, Event}
  Hom(𝓒): 态射集合
  ∘: 态射复合
  id: 恒等态射
```

### 1.2 三种范式的范畴构造

| 范式 | 范畴构造 | 核心函子 |
|------|---------|---------|
| **Virtual DOM** | 自由范畴 Free Category | `F: Set → Tree` |
| **Signals** | 图范畴 Graph Category | `Γ: State → ReactiveGraph` |
| **编译时模型** | 语法范畴 Syntax Category | `C: Lang → DOM` |

### 1.3 框架演进的函子序列

```text
                    [范畴 𝓢: 状态范畴]
                         │
                         ▼
              ┌─────────────────────┐
              │ 函子 F_react: 𝓢 → 𝓥  │
              │  (状态 → VDOM 树)    │
              └──────────┬──────────┘
                         │
                         ▼
              ┌─────────────────────┐
              │ 函子 F_solid: 𝓢 → 𝓖  │
              │  (状态 → 依赖图)      │
              └──────────┬──────────┘
                         │
                         ▼
              ┌─────────────────────┐
              │ 函子 F_svelte: 𝓢 → 𝓓  │
              │  (状态 → DOM 操作)    │
              └─────────────────────┘

自然变换 η: F_react ⇒ F_solid 不存在!
  ∵ VDOM 是树结构, Signals 是图结构
  ∴ 二者不是自然等价的
```
